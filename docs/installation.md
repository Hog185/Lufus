# Installation
## Arch Linux
To install Lufus on Arch Linux, you can use an AUR helper to grab the `lufus` package from the AUR. You can install it like this, given `paru` is your AUR helper of choice:
```bash
paru -S lufus
```

> [!CAUTION]
> The AUR package may not be in sync with the latest release of Lufus. 

## Other Distros
For other distros, we recommend following the build instructions.
